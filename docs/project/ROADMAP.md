<details>
<summary>v2.0.0 Standalone MCP Server - SHIPPED 2026-02-14</summary>

Extracted from trac_assist v1.3.2 as standalone package.

**Delivered:**
- 26 MCP tools (tickets, wiki, milestones, sync, system)
- Wiki page resources via MCP resource protocol
- Markdown <-> TracWiki format conversion
- Document sync tools
- stdio transport for MCP client integration
- Environment variable configuration
- Independent test suite

**Entry point:** `trac-mcp-server`
**Install:** `pip install .` or `pipx install .`
